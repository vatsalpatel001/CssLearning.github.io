  1. If grandchild has position: absolute, it will position itself relative to the browser window because there is no parent with a position other than the default of static.
  2. If parent also has position of relative, absolute, or fixed, grandchild will position itself relative to the boundaries of parent.
  3. However, if child also has a position of relative, absolute, or fixed, the grandchild will position itself relative to child's boundaries, because it is the nearest parent with a position other than static.
  4. In summary, relative affects an element's children, while absolute and fixed affect the element itself as well as its children.
  5. And remember that position: fixed bypasses all relative and absolute parents and always positions itself relative to the browser window.
